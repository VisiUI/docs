### Usage Methods

1. **Externalised Configuration**: Externalises footer data into a JSON or API file, making the component more flexible and easily updatable without declaring the data on the call.
2. **Loading States**: Add handling for loading states if the footer needs to retrieve data from an API, to improve the user experience.
3. **Event Handlers**: Add support for handling click events on links, allowing custom functions to be performed when they are clicked.
